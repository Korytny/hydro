# Hydro Genium 💧

Сайт производителя водородной воды в стеклянных бутылках и алюминиевых банках.

**Водородная вода — путь к долголетию, красоте и здоровью.**

## Технологии

- **React 19** + **Vite 8**
- **Tailwind CSS v4** — дизайн-система
- **Framer Motion** — анимации
- **React Router** — маршрутизация

## Структура

```
src/
├── components/     # UI-компоненты (Header, Hero, Benefits, Products...)
├── pages/          # Страницы (Home, Product, Science, Contacts)
├── lib/            # Утилиты (cn)
├── App.jsx         # Роутинг
└── main.jsx        # Точка входа
```

## Запуск

```bash
npm install
npm run dev        # дев-сервер
npm run build      # сборка в dist/
```

## Дизайн

Вдохновлён стилем **Air** (refero.design) — небесный фон, матовое стекло (frosted glass), воздушная типографика.
Адаптирован под бренд Hydro Genium: глубокий океанский синий, циановые акценты.
